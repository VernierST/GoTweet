System Requirements

Go!Tweet requires:
  • Vernier LabQuest, LabQuest Mini. or Go! devices (Go!Temp, Go!Link, or Go!Motion)
  • Internet Connection
  • Free Twitter account. If you do not have a Twitter account, you can sign up ap: http://twitter.com/signup
  • .NET Framework 3.5 SP 1 (Included in installer)
    • 1 GHz Pentium Processor (or equivalent)
    • 256 MB RAM
    • At least 500 MB available hard drive space
  • Windows Server 2003, Windows Server 2008, Windows XP, Windows Vista, or Windows 7
