The Options Dialog



Adjust your settings in the options dialog. From here, you can authorize Go!Tweet to connect to your Twitter account, set the interval of how often you want to tweet, and more.

Authorize Button: Use this button to link your Twitter account with Go!Tweet. You will not be able to fully use Go!Tweet until your account has been authorized. For more information, see Connecting With Twitter.

Tweet Every Text Box: Enter how often you want to update Twitter with a sensor reading. The timer starts as soon as Done is clicked.

Check the Only tweet when value is box to specify a threshold that is above, below, or equal to the value specified. NB: The threshold wiill only be checked at the interval previously specified.

Tweet Text: Provide a custom message to post when tweets are sent. Any occurence of <name> will be replaced by the name of the sensor as seen in the main window. Any occurence of <value> will be replaced with the reading from the sensor when the tweet was sent. Any occurence of <units> will be replaced by the units of the sensor as seen in the main window.
