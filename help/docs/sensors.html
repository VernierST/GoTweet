Supported Sensors

Go!Tweet supports most sensors that can plug into an analog channel on a LabQuest, LabQuest Mini. Go!Tweet only supports one sensor at a time. Go!Tweet also works with the Go!Temp, Go!Motion, and Go!Link.

There are a handful of sensors known to NOT work with Go!Tweet:

Sensors that require calculated columns:
Hand-Grip Heart Rate Monitor, order code HGH-BTA, $119.00
Spirometer, order code SPR-BTA, $199.00
Blood Pressure Sensor, order code BPS-BTA, $105.00
Flow Rate Sensor, order code FLO-BTA, $129.00

Sensors that do not have DDS Memory
Note: These sensors will work with a Go!Link
Stainless Steel Temperature Probe, order code TMP-BTA, $29.00
EKG Sensor, order code EKG-BTA, $147.00
Differential Voltage Probe, order code DVP-BTA, $39.00
Voltage Probe, order code VP-BTA, $12.00
Current Probe, order code DCP-BTA, $39.00

Sensors that require calibration before use:
Ammonium Ion-Selective Electrode, order code NH4-BTA, $179.00
Chloride Ion-Selective Electrode, order code CL-BTA, $179.00
Nitrate Ion-Selective Electrode, order code NO3-BTA, $179.00
Calcium Ion-Selective Electrode, order code CA-BTA, $179.00
